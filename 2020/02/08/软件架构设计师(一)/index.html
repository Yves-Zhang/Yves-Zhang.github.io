第四课 Cache 概念

Cache 的功能：

1.提高CPU数据输入输出的频率，突破冯·诺依曼瓶颈，即CPU与存储系统间数据传送宽带限制。

2.在计算机的存储系统体系中，Cache是访问速度最快的层次。

3.使用Cache改善系统性能的依据是系统的局部性原理。

例如：
Cache的命中率为 95%
Cache的周期时间为 1ns
主存储器周期时间为 1s
那么，系统读操作的平均周期时间为：
95%×1ns+(1－95%)×1000ns = 50.95ns
